To run programm in testing mode you need to start executable with {\ttfamily -\/-\/self-\/check} flag.

If you are using build\+\_\+and\+\_\+run.\+sh script you need to pass {\ttfamily -\/-\/self-\/check} flag after {\ttfamily -\/a} flag.

Example\+: {\ttfamily ./build\+\_\+and\+\_\+run.sh ./\texorpdfstring{$\ast$}{*}.cpp -\/a -\/-\/self-\/check}\hypertarget{md_README_autotoc_md6}{}\doxysubsection{\texorpdfstring{Editing tests}{Editing tests}}\label{md_README_autotoc_md6}
All tests are located at tests.\+txt for standard version and tests\+\_\+parse.\+txt for parsing version.

Each line consists of test before {\ttfamily @} and answers after {\ttfamily @}. First number after {\ttfamily @} symbol stands for number of solutions. This number may be {\ttfamily 0}, {\ttfamily 1}, {\ttfamily 2} or {\ttfamily -\/1}(infinite).
